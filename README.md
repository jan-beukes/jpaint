# Jpaint
## A Simple paiting application focused on pixel art

![](screenshot.png)

## Controls

| Key    | Control      |
| ------ | ------------ |
| b      | brush        |
| e      | eraser       |
| h      | hand         |
| g      | paint bucket |
| c      | clear canvas |
| q      | color select |
| Mouse-scroll | resize brush |
| Ctrl+scroll  | zoom canvas  |
| middle-mouse | pan          |
| L-Alt  | color picker |
| Ctrl+z | Undo         |
| Ctrl+y | Redo         |
| Ctrl+s | Save         |
| Ctrl+o | Open         |
| Ctrl+n | new canvas   |


## Features
- [x] Save and load Images
- [x] Create new custom canvas 
- [x] paint bucket
- [x] color picker
- [x] undo/redo
